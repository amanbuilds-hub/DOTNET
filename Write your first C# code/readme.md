### Write you first code

**Console.Write("Congratulations!!);**

Diff between WriteLine and Write : 
- WriteLine : Display the output on the console and shift the pointer on the nextLine.
- Write : Display the output on the console only.

*you saw that the message Hello World! was printed to the output console. When the phrase is surrounded by double-quotation marks in your C# code, it's called a **literal string**.*

The Console part is called a **class**.
